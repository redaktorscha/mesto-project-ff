# Проектная работа Mesto

## Приложение для обмена фотографиями

---

![Проектная работа Mesto](https://github.com/redaktorscha/mesto-project-ff/blob/843424859e1e58dafd744a76983d0ba4b93bdd42/src/images/mesto_project.png)

### О проекте

Сайт, где можно выкладывать фотографии различных локаций с кратким описанием и ставить лайки.
Автор страницы может поменять аватарку и отредактировать данные о себе.

### Использованные технологии

- HTML 5
- CSS
- JS
- Webpack (сборка)

### Деплой

[GitHub Pages](https://redaktorscha.github.io/mesto-project-ff/)

### Собрать и развернуть локально

- npm run dev - режим разработки
- npm run build - продакшен-версия
